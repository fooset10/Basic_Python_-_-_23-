1. Переменная в программировании — это именованная область памяти компьютера, предназначенная для хранения данных, которые могут изменяться в ходе выполнения программы. Она позволяет сохранить какое-то значение (число, текст, результат вычислений) и использовать его позже.

gist.github.com
Особенности переменных:
Имеют имя (идентификатор).
Значение — информация, которая хранится в памяти и с которой работает программа.
Тип — определяет характер значений, принимаемых переменной, виды операций с ними и их представление в памяти компьютера.
 
skillbox.ru
blog.skillfactory.ru
Виды
Некоторые виды переменных в программировании:
Числовые — хранят числа: целые (1, 42, -10) и дробные (3.14, 0.5).
Текстовые (строки) — хранят буквы, слова и предложения.
Логические — хранят True или False.
 
progkids.com
Также переменные могут быть:
Статическими — тип данных определяется на этапе компиляции программы, заранее задаётся программистом.
Динамическими — тип переменной определяется на этапе присвоения значения (то есть исполнения программы), а не его объявления. Благодаря этому в различных участках программного кода одна и та же переменная может принимать значения разных типов.
 
blog.skillfactory.ru
Объявление
Переменные объявляют с определённым именем и типом. Инициализация — это присвоение переменной начального значения. 
sky.pro
Некоторые правила при объявлении переменных:
Имя должно быть описательным и отражать назначение переменной (например, userName вместо x).
Стиль именования — следует принятым в языке конвенциям (camelCase в JavaScript, snake_case в Python и т. д.).
Ограничения: имена не должны начинаться с цифры, содержать пробелы или специальные символы (кроме _).
Нельзя использовать ключевые слова языка как имена переменных.
 
sky.pro
Область видимости
Область видимости (scope) переменной — это часть программы, в которой переменная доступна для использования. Некоторые типы областей видимости: 
python-academy.org
Глобальная — переменные объявлены на самом верхнем уровне, вне каких-либо конструкций, и доступны из любой части кода.
Блочная — переменные объявлены внутри блоков кода, например инструкций if и циклов, и доступны только внутри этих блоков.
Область видимости функции — переменные объявлены внутри функций и доступны только внутри них.
 
netology.ru
Области видимости входят друг в друга, формируя иерархию. 
netology.ru
Ошибки
Некоторые ошибки, связанные с неправильным использованием переменных:
Неправильное присвоение — присваивание значения переменной, несовместимого с её типом. Например, попытка присвоить строковое значение целочисленной переменной вызывает ошибку.
Переполнение переменной — некоторые типы данных имеют ограничение по размеру, если значение превышает допустимый диапазон, это вызывает переполнение.
Неинициализированные переменные — использование переменных без предварительного задания начального значения ведёт к ошибкам, так как они могут содержать случайные данные.



2. Функция в программировании — это именованный блок кода, который выполняет определённую задачу и может быть вызван из других частей программы. Можно представить функцию как «чёрный ящик»: в неё передают входные данные, она выполняет с ними операции и возвращает результат.

Технология программирования функций
Структура
Базовая структура функции обычно включает следующие элементы: 
sky.pro
Имя — уникальный идентификатор функции. Должно отражать суть действия, которое выполняет функция.
Параметры — входные данные, которые функция принимает (могут отсутствовать).
Тело функции — код, который будет выполнен при вызове. Эти инструкции определяют, что делает функция.
Возвращаемое значение — результат, который функция возвращает после своего выполнения. Не все функции возвращают значение — некоторые просто выполняют действие и завершаются.
